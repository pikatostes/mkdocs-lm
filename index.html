Bienvenido a mi página creada con MKDocs para la asignatura Lenguaje de Marcas

Proceso de creación

MKDocs es una herramienta de código abierto que nos permite crear documentación en formato HTML a partir de archivos Markdown. En este texto, explicaremos cómo crear una página web utilizando MKDocs, desde la instalación hasta la generación del sitio.

Para comenzar, es necesario instalar MKDocs. Esto se puede hacer utilizando el gestor de paquetes pip de Python:

pip install mkdocs

Una vez instalado, podemos crear un nuevo proyecto con el comando mkdocs new:

mkdocs new my-site
cd my-site

Esto creará una estructura de archivos básica para nuestro sitio. El archivo mkdocs.yml es el archivo de configuración principal, donde se pueden especificar detalles como el título del sitio, la lista de páginas, etc. Además, en la carpeta docs es donde colocaremos todos los archivos Markdown que conformarán la documentación de nuestro sitio.

Podemos crear una página nueva simplemente creando un archivo Markdown en la carpeta docs. Por ejemplo, podríamos crear una página llamada index.md que contenga el siguiente contenido:

# Bienvenidos a mi sitio

Este es mi primer sitio web creado con MKDocs.

Una vez que hayamos creado nuestras páginas, podemos generar el sitio HTML utilizando el comando mkdocs build:

mkdocs build

Esto generará una carpeta site que contiene todo el sitio web HTML. Podemos abrir el archivo index.html en nuestro navegador para ver el sitio.

También podemos servir el sitio localmente con el comando mkdocs serve:

mkdocs serve

Esto iniciará un servidor web local en http://localhost:8000 donde podemos ver nuestro sitio en tiempo real mientras trabajamos en él.

Además, MKDocs ofrece una serie de temas y complementos que podemos utilizar para personalizar nuestro sitio. Estos se pueden especificar en el archivo de configuración mkdocs.yml. Por ejemplo, si queremos utilizar el tema material y el complemento search, podemos agregar las siguientes líneas a mkdocs.yml:

theme:
    name: material

plugins:
    - search

Con esto, podemos crear un sitio web completo de documentación utilizando MKDocs, aprovechando todas las herramientas y funcionalidades que nos ofrece.
